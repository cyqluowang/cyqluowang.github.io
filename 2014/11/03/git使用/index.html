跟新代码

假如本地已经存在了这个项目，而仓库中又有一新的更新，如何把更的合并到本地的项目中？

git pull origin gh-pages
等于
git fetch origin gh-pages
//将origin/gh-pages合并到当前分支
//origin/gh-pages 远端分支名
git merge origin/gh-pages
